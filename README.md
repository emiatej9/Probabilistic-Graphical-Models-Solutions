## Chapter 2
### Exercise 2.1
Prove the following properties using basic properties of definition 2.1:

a. P(∅) = 0

```
P(Ω) = P(Ω ∪ ∅)
     = P(Ω) + P(∅) (by definition 2.1)
     = 1
     
∴ P(∅) = 0
```
